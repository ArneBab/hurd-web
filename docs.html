Introductory material

  • Towards a New Strategy of OS Design, an architectural overview by Thomas Bushnell, BSG.
  • The Hurd, a presentation by Marcus Brinkmann.
  • GNU/Hurd User's Guide, an introduction to the important concepts and software of the GNU system, written for new users, AKA "GNUbies."

    Available Formats:

    • HTML version for browsing online.
    • PostScript version [477kB, 67 pages] for download.
    • ASCII text version [154kB].
    • Texinfo source [155kB].

    Available Formats:

    • HTML version for browsing online.
    • PostScript version [187kB, 37 pages] for download.
    • ASCII text version [59kB].
    • Texinfo source [60kB].

Reference manuals

The GNU Hurd Reference Manual documents the architecture, the usage and the programming of the GNU Hurd. At the moment, the manual is quite incomplete.

Available Formats:

  • HTML version for browsing online.
  • PostScript version [1020kB, 91 pages] for download.

If you want to work on the manual, you may also download the Texinfo sources (hurd.texi [193k], gpl.texinfo [18k], and version.texi). Then, you should submit any modifications to bug-hurd@gnu.org in unidiff format (produced by diff -u).
